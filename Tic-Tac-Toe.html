Tic-Tac-Toe Using Java Swing

Project Description

Developed a simple Tic-Tac-Toe game using Java and Swing for GUI-based interaction.

Key Features

  • Implemented a fully functional two-player game mode.
  • Designed an intuitive graphical user interface (GUI) using Java Swing.
  • Incorporated game logic and validation for win conditions.

Technologies Used

Java
Java Swing
